# Faradey-android


## Описание

Faradey для Аndroid  - мобильное приложение, которое сочетает в себе безопасность и конфиденциальность общения с эффективным управлением проектами. Это обеспечивает пользователям гибкость в управлении задачами, при этом гарантируя сохранность их данных от утечек в сеть.
